# Image Requirements for Zinovia Website

All images should be placed in the `/public/images/` directory. Please provide the following:

## Essential Images (Required)

### 1. Logo
- **File**: `logo.svg` or `logo.png`
- **Location**: `/public/logo.svg` or `/public/logo.png`
- **Dimensions**: 
  - SVG: Scalable (preferred)
  - PNG: Minimum 200x60px (for header), 400x120px (for footer)
- **Format**: SVG (preferred) or PNG with transparent background
- **Usage**: Header, Footer, Favicon, Open Graph

### 2. Favicon
- **File**: `favicon.ico`
- **Location**: `/public/favicon.ico`
- **Dimensions**: 32x32px, 16x16px (multi-size .ico)
- **Format**: .ico
- **Usage**: Browser tab icon

### 3. Open Graph Image (Social Media Sharing)
- **File**: `og-image.jpg` or `og-image.png`
- **Location**: `/public/og-image.jpg`
- **Dimensions**: 1200x630px (required for social media)
- **Format**: JPG or PNG
- **Usage**: When sharing on social media (Facebook, Twitter, LinkedIn)

## Hero Section Images

### 4. Hero Background Image (Optional)
- **File**: `hero-background.jpg` or `hero-background.webp`
- **Location**: `/public/images/hero-background.jpg`
- **Dimensions**: 1920x1080px or larger
- **Format**: JPG, WebP (optimized)
- **Usage**: Hero section background
- **Note**: Currently using subtle pattern - can add image if desired

## Image Guidelines

- **Optimization**: All images should be optimized for web (compressed)
- **Naming**: Use lowercase with hyphens (kebab-case)
- **Alt Text**: Will be added in components for accessibility
- **Formats**: 
  - Photos: WebP (preferred) or JPG
  - Logos/Icons: SVG (preferred) or PNG
  - Simple graphics: SVG
