# Zinovia Logo Design Guide

## Overview

The Zinovia logo represents the company's core identity as an **Enterprise AI Solutions Platform**. The design combines:

- **Neural Network Icon**: Represents AI intelligence, machine learning, and automation
- **"Z" Shape**: Forms the letter Z from Zinovia using neural network connections
- **Modern Gradient**: Purple-to-cyan gradient reflecting innovation and technology
- **Professional Typography**: Clean, bold Inter font for enterprise credibility

## Logo Files

### Main Logos
- **`logo.svg`** - Full logo with text (light background version)
- **`logo-dark.svg`** - Full logo with text (dark background version)
- **`logo-icon.svg`** - Icon-only version (standalone)

### Favicon
- **`favicon.svg`** - Simplified icon for browser tabs (32x32)

## Design Elements

### Colors
- **Primary Purple**: `#8E54FF` (rgb(142, 84, 255)) - Brand identity
- **Accent Cyan**: `#5BF5FF` (rgb(91, 245, 255)) - Innovation & technology
- **Text Dark**: `#121928` (light mode) / `#F5F9FF` (dark mode)

### Icon Concept
The neural network icon forms a "Z" shape through interconnected nodes and pathways, symbolizing:
- **AI Intelligence**: Neural network nodes represent machine learning
- **Connectivity**: Connections show integration and automation
- **Enterprise Solutions**: Clean, professional aesthetic
- **Innovation**: Modern gradient suggests cutting-edge technology

### Typography
- **Font**: Inter (system fallback: -apple-system, BlinkMacSystemFont, sans-serif)
- **Weight**: 700 (Bold) for "Zinovia"
- **Size**: 28px for main text, 14px for ".AI" badge
- **Letter Spacing**: -0.02em for main text, 0.05em for badge

## Usage Guidelines

### Light Backgrounds
Use `logo.svg` on light/white backgrounds

### Dark Backgrounds
Use `logo-dark.svg` on dark backgrounds (footer, dark mode)

### Icon Only
Use `logo-icon.svg` when space is limited or for app icons

### Minimum Sizes
- Full logo: Minimum 140px width
- Icon only: Minimum 40px × 40px
- Favicon: 32px × 32px

### Spacing
Maintain clear space around the logo equal to at least 20% of the logo height

## Brand Alignment

The logo aligns with Zinovia's brand values:
- ✅ **Enterprise-Grade**: Professional, trustworthy design
- ✅ **AI-Focused**: Neural network icon clearly represents AI
- ✅ **Innovation**: Modern gradient and clean aesthetics
- ✅ **Speed**: Dynamic connections suggest rapid deployment
- ✅ **Intelligence**: Neural network represents smart automation

## Technical Specifications

- **Format**: SVG (scalable vector graphics)
- **ViewBox**: Optimized for scaling at any size
- **Gradients**: Linear gradients for smooth color transitions
- **Accessibility**: High contrast ratios for readability
- **Performance**: Lightweight SVG files for fast loading

Update these components to use the new logo files as needed.
